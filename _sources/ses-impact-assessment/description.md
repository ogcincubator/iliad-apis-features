# SeaDOTs SES Impact Assessment

This draft building block describes an assessment case built around a socio-ecological-system model.

The key contract is traceability:

`model -> indicators -> quality specifications -> input data -> experiments -> outputs -> relationships`

It is intended for cases such as the Norwegian Utsira Nord floating-wind assessment, where WindFloat Atlantic biodiversity monitoring is used as a reference case but must be transferred cautiously through explicit quality gates.

The block does not replace ODD, OIM variables, OIM observations, or property relationships. It coordinates them:

- ODD describes the assessment model and quality-aware process.
- OIM variables define indicator/observed-property URIs.
- Indicator quality requirements define readiness gates.
- Property relationships define weighted indicator links.
- Input/output data assessments connect inventory records to indicators and products.

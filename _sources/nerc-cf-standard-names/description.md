## NERC CF Standard Names

This building block profiles the NERC Vocabulary Server Climate and Forecast Standard Names collection (`P07`) as a reusable semantic asset for ILIAD metadata and observation workflows.

It does not duplicate the full controlled vocabulary inside this repository. Instead, it defines the expected RDF and SHACL structure for referencing NERC CF concepts, with examples that use authoritative NERC term URIs.

The block is intended for cases where datasets, variables, observed properties, or metadata records need stable links to CF standard names managed through the NERC Vocabulary Server.
